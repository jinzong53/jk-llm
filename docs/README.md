# JK-LLM 文档目录

本目录用于面向学习者的分章文档，直接从大语言模型的核心技术部分切入：注意力、Transformer、模型结构、分词、嵌入与损失函数等。文档目标是扎实、可读并结合本项目代码给出实例。

章节列表（后续会逐步补充，每写完一章会暂停让你检查）：

1. 01-attention.md — 注意力机制（本章已完成草稿）
2. 02-transformer.md — Transformer（decoder-only 聚焦）
3. 03-model-structures.md — 大模型常用结构与变体
4. 04-tokenization-embeddings.md — 分词器与嵌入
5. 05-losses-training.md — 损失函数与训练目标
6. 06-practical-tips.md — 训练与调优实践要点（包括本项目中的配置说明）

阅读建议：按顺序阅读每章，理论结合 `src/` 代码观察实现细节。欢迎在每章后反馈改进意见。
